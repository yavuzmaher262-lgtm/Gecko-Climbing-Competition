Cara Deploy Live Score (HTML Statis)

1. Upload file index.html ke hosting atau server.
2. Jika memakai cPanel: File Manager → public_html → upload index.html.
3. Jika memakai GitHub Pages:
   - Buat repo → upload index.html → Settings → Pages → pilih branch → Save.
4. Untuk data live, kamu bisa menambahkan:
   - Fetch API
   - Websocket
   - Spreadsheet backend (ex: Airtable, Supabase)

File ini hanya versi dasar untuk tampilan awal.
